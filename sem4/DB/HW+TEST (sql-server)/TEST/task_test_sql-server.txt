У базі даних кадастру зберігаються відомості про прямокутні земельні ділянки, їхніх власників та час набуття власності. Ділянка визначається географічними координатами південно-західного кута, а також розмірами. Про власника відомі ПІБ, рік народження, паспорт. Він може володіти багатьма ділянками. Ділянки не можуть перетинатися. Набувати право власності можуть лише особи, не молодші 18 років.
